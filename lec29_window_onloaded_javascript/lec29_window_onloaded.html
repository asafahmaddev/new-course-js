Window Onload Example

In JavaScript, `window.onload` is an event handler that executes a function when the entire content of a page (including images, scripts, and stylesheets) has finished loading. This is useful for running code that relies on the DOM being fully loaded.

In JavaScript, `window.onload` is an event handler that executes a function when the entire content of a page (including images, scripts, and stylesheets) has finished loading. This is useful for running code that relies on the DOM being fully loaded.

An alternative event is `DOMContentLoaded`, which triggers when the HTML document has been completely loaded and parsed, without waiting for stylesheets, images, and subframes to finish loading.
